# Programming for Business (2025/2026)

This repository contains all teaching materials, code, and exercises for the **Programming for Business** course for the academic year **2025/2026**.

It is organized into two main parts: **R** and **SAS**, covering different stages of data analysis and programming concepts used throughout the course.

## ⚙️ Requirements

### R Section
- R (latest version recommended)
- RStudio

### SAS Section
- Access to **SAS OnDemand for Academics**
  - All code is written and tested to run in this environment
  - No local SAS installation is required

---

## ▶️ How to Use This Repository

If you are working on RStudio for the R part you just need to:

1. Clone or download this repository.
2. Open the folder if you have made the download of it in the RStudio environment.
3. Explore the documents and exercises in the files.

For the SAS part it is necessary to:

1. Download the repository.
2. Upload the Files to SAS OnDemand for Academics.
3. Change the paths of the files to you relative paths.
4. Explore the files and exercises as you wish

---

## 📝 Notes

- The SAS code is specifically designed for **SAS OnDemand for Academics**, so minor adjustments may be needed for other environments.
- Each chapter builds on the previous one.
- Students are encouraged to attempt exercises on their own before consulting the solution files.

---

## 📌 Learning Objectives

By working through this repository, students will:

- Develop foundational programming skills in **R**
- Gain practical experience with **SAS for data manipulation and reporting**
- Learn how to access, clean, combine, and analyze datasets
- Produce structured reports from raw data
- Integrate R coding into a collaborative coding platform like GitHub
